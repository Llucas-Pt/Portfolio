SOBRE MIM

Meu nome é Lucas Vicente, e minha jornada como desenvolvedor começou com uma graduação em Análise e Desenvolvimento de Sistemas pela Unopar, complementada por uma formação técnica em Redes de Computadores pela Vasco Coutinho. Minha carreira inicialmente focada em infraestrutura me proporcionou uma base sólida em suporte ao usuário, administração de banco de dados e administração de servidores.

Com o tempo, meu fascínio pela capacidade de criar e inovar através do desenvolvimento Web me levou a uma transição de carreira. Dediquei-me a dominar tecnologias front-end e back-end, incluindo Javascript, React, HTML, CSS, Node e SQL. Essa mudança não apenas reflete meu interesse profundo por tecnologia, mas também minha busca constante por desafios e meu compromisso com o aprendizado contínuo.

Apaixonado por solucionar problemas e otimizar processos, hoje me concentro em contribuir para a evolução de produtos, utilizando minha expertise técnica para impactar positivamente os projetos em que me envolvo. Minha trajetória é marcada por uma transição consciente para o desenvolvimento, guiada por uma paixão incansável pela tecnologia e um desejo de estar na vanguarda da inovação digital.

PROJETOS


SPACEX


Este projeto representa a reconstrução meticulosa do website anterior da SpaceX, uma iniciativa que me permitiu mergulhar profundamente na aplicação prática de meus conhecimentos adquiridos em Bootstrap, além de aprimorar minhas habilidades em HTML e CSS. O objetivo central deste desenvolvimento foi não apenas replicar visualmente o site, mas também compreender e implementar as funcionalidades dinâmicas que o Bootstrap oferece, garantindo uma experiência de usuário fluída e responsiva. Através deste desafio, tive a oportunidade de explorar as capacidades deste framework de front-end, reforçando minha base técnica e contribuindo para o meu crescimento profissional na área de desenvolvimento web.

Landing


Este projeto é uma demonstração eloquente do meu aprofundamento técnico e criativo no uso de HTML e CSS para a elaboração de landing pages. Com foco em design responsivo e aprimoramento da experiência do usuário, esta iniciativa reflete meu compromisso em dominar as fundamentações do desenvolvimento web. Através da construção meticulosa de cada elemento e da implementação de práticas de codificação modernas, busquei não apenas criar páginas web esteticamente agradáveis, mas também funcionalmente robustas.

SISTEMA DE CADASTRO


Este projeto representa uma solução avançada e integrada para o gerenciamento de cadastros de funcionários, elaborado com o uso de tecnologias de ponta como Node.js, Express, Sequelize, MySQL, HTML e CSS. O sistema foi meticulosamente projetado para facilitar o processo de inclusão de novos funcionários na base de dados, oferecendo uma interface amigável e intuitiva para o usuário final. Além do cadastro, o sistema permite a visualização detalhada dos dados cadastrados, proporcionando uma ferramenta completa para a gestão de recursos humanos diretamente do navegador. Através da implementação de práticas modernas de desenvolvimento web, este projeto visa otimizar os processos internos de empresas, garantindo eficiência e precisão no gerenciamento de informações de funcionários
